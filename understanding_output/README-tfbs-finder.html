These two subdirectories contain TFBS detection results for Arabidopsis thaliana using two restrictive PWM (Position Weight Matrix) sets. Each set corresponds to a curated subset of transcription factor motifs derived from the ConnectF dataset and validated experimentally. The details and curation process for these motifs are documented in the Zenodo record: 10.5281/zenodo.17144325.
